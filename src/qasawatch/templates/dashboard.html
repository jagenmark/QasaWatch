Gmail setup
  1. Use an existing Gmail address as the sender. It can also be a recipient.
  2. Turn on 2-Step Verification for that Google account.
  3. Create a Google App Password named “QasaWatch”.
  4. Provide that 16-character password as QASAWATCH_SMTP_PASSWORD. For local use, put it in .env. Restart QasaWatch, then use “Test email” below.
